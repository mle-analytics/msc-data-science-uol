# Pokemon Card Market Analysis
An NLP and Exploratory Data Analysis (EDA) project focused on price trends and keyword frequency in Pokemon card listings.

## Overview
This project processes a dataset of Pokemon cards to identify:
*   **Price Metrics:** Statistical summary of card prices and image counts.
*   **Rarity Impact:** Analysis of how rarity classes affect the average market price.
*   **NLP Insights:** Keyword frequency analysis of card titles using NLTK to identify market trends.

## Tech Stack
*   **Language:** Python 3.x
*   **Libraries:** Pandas (Data Manipulation), NLTK (Natural Language Processing), Matplotlib (Visualization).

## How to Run
1. Clone the repo.
2. Install dependencies: `pip install -r requirements.txt`.
3. Run `python analysis.py`.
